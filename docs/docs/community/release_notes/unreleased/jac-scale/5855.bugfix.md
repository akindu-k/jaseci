- **Fix: topology index updates no longer silently dropped by MongoBackend sync**: when a walker only modified `topology_index_data`, `MongoBackend.sync` skipped the write due to an unchanged hash. The sync now checks the `is_topology_dirty` flag and bypasses both the hash early-out and the `put_partial` path (which only writes `data.archetype.*` fields), falling through to a full `_write_to_db` upsert that includes `data.topology_index_data`.
