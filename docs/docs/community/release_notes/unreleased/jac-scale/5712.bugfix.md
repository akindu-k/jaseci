- **Fix: SqliteMemory.sync() no longer checks write access on unchanged anchors**: Previously, `sync()` would call `Jac.check_write_access()` unconditionally on every anchor, causing read-only walkers to emit access-denied logs even when persisting unchanged data. The access check is now gated on actual changes (dirty fields or access modifications), preserving the invariant that read-only walkers never trigger the write path during synchronization.
